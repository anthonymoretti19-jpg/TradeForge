# TradeForge Pro

All-in-one app for mechanical trades - steamfitting, pipefitting, gas work, and HVAC.

## Features (v1.0)

### Fabrication
- Fitting allowances (45° and 90°)
- Offset calculator
- Rolling offset calculator
- Pipe charts (ASME standards)

### Gas Work
- Natural gas and propane sizing
- BTU calculations

### Job Management
- Save jobs with photos and notes
- Material takeoffs
- Customer information

### Offline First
- Works completely offline with SQLite
- Dark mode UI

## Tech Stack
- React Native + Expo
- TypeScript
- SQLite (offline storage)

## Getting Started

```bash
npm install
npm start
```

Run on iOS/Android via Expo Go app.
